# Example Datasets

This repository contains example datasets used throughout articles and tutorials on my website. The datasets are intended for demonstrating data cleaning, analysis, visualisation, and biological data workflows.

Most datasets are hypothetical and were created for educational purposes.

## Available Datasets
### Species Observation Data

A hypothetical species observation dataset used to demonstrate common stages of a biological data workflow, including:

Data cleaning
Handling missing values
String manipulation
Pivot tables
Species abundance analysis
Species richness analysis
Bar charts
Choropleth maps
Symbol maps
Interactive dashboards

The following files are currently available:

File|Description
---|---
raw_species_observations.csv|Original dataset containing inconsistencies and missing values.
cleaned_species_observations.csv|Cleaned dataset with standardised values and geographical information.
species_abundance.csv|Summary table showing the number of observations recorded for each species.
species_richness_by_region.csv|Summary table showing the number of unique species recorded in each region.
species_richness_by_site.csv|Summary table showing the number of unique species recorded at each survey site together with their coordinates.

## Intended Use

These datasets are provided to allow readers to follow along with tutorials and reproduce the examples presented in the articles.

The datasets are hypothetical and do not represent actual survey data.
